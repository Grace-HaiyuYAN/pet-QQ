# AI 办公宠物助手 —— 设计文档

## 一、问题诊断

### 核心痛点
企业办公管理面临"效率与人文"的两难困境：
1. **管理者视角**：需要员工行为数据支撑决策（任务进度、工作状态、团队协作），但缺乏非侵入式的数据采集手段
2. **员工视角**：反感被监控，传统考勤/任务系统带来压迫感，降低工作积极性和归属感
3. **HR 视角**：难以及时发现员工倦怠、离职风险，人文关怀缺乏数据支撑；入离职流程重复繁琐

### 问题本质
管理的核心在于获取数据，但数据采集与员工体验存在天然矛盾。

## 二、方案设计

### 核心理念
**"隐形管理 + 人文关怀"** —— 让管理不可见，让关怀可感知

### 产品形态
复活 QQ 宠物，接入 Agent 能力，打造桌面 AI 办公助手

### 双端架构

**员工端（桌面宠物）**
- 可爱虚拟宠物形象，桌面常驻陪伴
- 双击展开功能面板：对话框、今日 Todo、日程提醒
- 三栏任务视图：任务拆解 | 知识库推荐 | 协作者推荐

**HR/管理端（数据看板）**
- **团队状态总览**：用不同宠物形象/状态表示员工当前状态（高效、疲劳、过载）
- **员工画像分析**：通过项目贡献、考勤、绩效数据生成能力画像，识别高潜人才与离职风险
- **AI 管理建议**：基于数据给出干预建议，管理者可选择采纳或忽略
- **流程自动化**：入离职流程自动化，减少 HR 重复操作
- **培训推荐**：根据员工技能差距，推荐个性化培训课程

### 核心功能

| 功能模块 | 员工感知 | 管理价值 |
|---------|---------|---------|
| 任务拆解 | AI 帮我分解任务 | 获取任务进度数据 |
| 日程自动化 | 自动提醒会议/截止日期 | 了解时间分配 |
| 知识库推荐 | 智能推荐相关文档 | 分析知识需求 |
| 协作者匹配 | 推荐在做类似事情的同事 | 促进团队协作 |
| 喝水/休息提醒 | 健康关怀 | 监测工作强度 |
| 社群活动 | 积分激励、黑客松 | 提升 AI 使用率与团队凝聚力 |
| 员工画像 | 不可见 | 发现高潜人才、识别离职风险 |
| 状态监测 | 收到关怀反馈 | AI 监测疲劳/过载并建议干预 |

### HR 看板详细设计

**1. 团队状态面板**
- 以宠物矩阵形式展示团队成员状态
- 不同宠物表情代表不同状态：开心(高效)、困倦(疲劳)、冒火(过载)、平静(正常)
- 点击单个宠物可查看该员工详情

**2. 员工画像卡片**
- 贡献度雷达图：业务产出、协作指数、学习成长、出勤稳定性
- 风险预警标签：离职风险、倦怠风险、过载风险
- 成长轨迹：入职以来的能力提升曲线

**3. AI 建议中心**
- 基于数据的管理建议列表
- 每条建议标注依据数据和置信度
- 管理者可标记"采纳/忽略/稍后处理"
- 示例建议：
  - "张三近两周任务完成率下降 30%，建议关注是否需要支援"
  - "李四连续 5 天工作超 10 小时，建议提醒休息"
  - "王五本月协作指数提升 50%，可考虑晋升评估"

**4. 人文关怀模块**
- 自动检测员工异常状态（长时间无操作、频繁加班、任务堆积）
- 触发关怀动作：宠物主动问候、推送休息建议、通知管理者
- 关怀记录：管理者可查看已触发的关怀及员工反馈

### 规避监控过度的机制

| 机制 | 具体措施 |
|-----|---------|
| 数据边界 | 只采集工作产出相关数据，不监控私聊、非工作时段 |
| 员工参与 | 匿名问卷投票决定考核指标，增加透明度 |
| 授权机制 | 员工可选择是否授权数据共享，可随时撤回 |
| 人工空间 | AI 建议仅作参考，最终决策由管理者结合实际做出 |

## 三、AI 工具选择

### 选型：Claude Code + Web 技术栈

**选择理由**：
1. **快速原型**：Claude Code 可一键生成完整前端 Demo
2. **交互丰富**：Web 技术支持动画、拖拽等宠物交互
3. **演示友好**：浏览器运行，无需安装，便于面试展示
4. **迭代高效**：自然语言描述即可修改功能

## 四、关键配置

### MVP 核心体验
1. 桌面宠物形象（可爱、会动）
2. 双击展开任务面板（员工端）
3. Tab 切换到 HR 看板（管理端）
4. 模拟 AI 对话与建议能力

### KPI 体系
- **业务贡献**：任务完成率、产出质量、响应速度
- **社区贡献**：社群活跃度、活动参与、AI 使用率

## 六、效果评估

### Demo 目标
- 清晰展示"桌面宠物 + AI 助手"的核心体验
- 体现"隐形管理"理念：员工端看不到监控感
- 展示双端价值：员工提效 + 管理洞察 + 人文关怀

### 预期效果
- 员工接受度提升：宠物形象降低抵触
- 管理效率提升：数据驱动决策
- 人文关怀落地：AI 监测 + 主动干预
- HR 效率提升：流程自动化 + 风险预警
